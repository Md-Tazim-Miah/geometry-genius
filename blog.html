1. What Is DOM And What Is The Purpose Of DOM?

DOM is a short form of Document Object Model. It is a programming interface web documents. When a web browser loads a webpage it becomes a DOM to manipulate the content, structure and style of the webpage by programming languages like JavaScript. In the DOM each element of the HTML language become node in the tree.
The purpose of the DOM is that it provides a way for scripts to dynamically access and update the content, structure, and style of a document. This capability is what makes modern, interactive web pages possible.

3. What Is Event Bubble?

When events propagate upward in the DOM tree to the parents and ancestors of elements that receive them. This is known as "Event Bubbling," and it gives parent elements the ability to manage events that happen on their child components.

4. What Is A Callback Function And Why Will You Use It?

A callback function is one that is called after another function has finished running or an event has occurred, and it is called as an argument to that other function.
I will use callback function when I have to manage asynchronous events or network requests. Because they serve as a way to ensure that specific code won't execute until a particular task has been finished.
